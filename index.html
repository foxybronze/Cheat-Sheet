HTML & CSS Cheat Sheet

HTML

HTML Tables

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The stable row container
<td> Table Data The stable cell container
<tfoot> Table Foot The set of rows defining the footer in a table.

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.

CSS

Syntax and Selectors

<link> Link Element

The Link Element is used to link HTML documents to external resources like CSS files.
Element Name Description
href HRef Attribute to specify the URL to the external resource
rel Relationship Attribute to specify the relationship of the linked document to the current document.
type Type Attribute to define the type of content being linked.

Class and ID Selectors

Selector Reference
type .e.g., h1
class .class
ID #ID
Selector Grouping e.g., h1, h2 (rule applies to all)
Selector Chaining e.g., h3.section-heading (rule for very specific selector)
